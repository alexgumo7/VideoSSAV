Introduction

Video SSAV is a university project focused on implementing a video streaming system. The system consists on three blocks: Do Something Block, Server and Streaming. We have focused on studying the Do Something Block and the Quantizer with Matlab.We have implemented the General Video Coding System with a system taken off the shelf. Later we implemented the server and the streaming.

System Description

General Video Coding System

Prepares the video in order to upload it to the internet. It is necessary to reduce its size and therefore it makes it possible to transmit it. We have used AVIDEMUX.

Server

We use an Apache server in order to use our own computer to host our web.
